Completely disables theme, plugin, and core updates, and disables file editing from the WordPress admin dashboard.

== Description ==

**Nexevo - Theme Update & File Edit Disabler** is a lightweight plugin that locks down your site by completely disabling:

- WordPress core updates
- Theme and plugin updates
- Automatic updates
- Plugin/theme installation or deletion
- File editing in the admin panel

Perfect for agencies, staging sites, production environments, or client websites.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/nexevo-theme-update-disabler/` directory.
2. Activate the plugin through the “Plugins” screen in WordPress.
3. No configuration needed. It works automatically.

== Frequently Asked Questions ==

= Will this plugin block security updates? =
Yes, it disables all WordPress updates including security. Use cautiously on public or high-traffic sites.

= How do I re-enable updates or file editing? =
Just deactivate the plugin from the “Plugins” screen.

= Is this compatible with WordPress Multisite? =
Yes. Network activate it to apply across all subsites.
